# Dev Circle

## 📌 Project Overview

Dev Circle is a web-based platform designed to connect developers, share ideas, and collaborate on projects. It enables users to interact, post content, and build a community-driven environment.

---

## 🚀 Features

* User authentication (Login / Signup)
* Create and share posts
* Interactive UI for community engagement
* Scalable backend architecture

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB

---

## 📊 Project Status

* ✅ Backend: Completed
* 🚧 Frontend: In Progress (approximately 50% completed)

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/Toufeeqmir/Dev-Circle.git
cd Dev-Circle
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env` file in the root directory and add:

```
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
```

### 4. Run the project

```bash
npm start
```
